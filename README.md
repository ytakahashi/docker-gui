# docker-gui

A simple graphical UI for docker.

![image](https://user-images.githubusercontent.com/26239560/45251940-a2526380-b389-11e8-9721-0da3c5224a94.gif)

## Getting Started

```console
git clone https://github.com/ytakahashi/docker-gui.git
cd docker-gui
yarn
```

### build an app

```console
yarn build
```
